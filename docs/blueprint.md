# **App Name**: Event Manager Pro

## Core Features:

- Admin Authentication: Enable secure login for admin users using Firebase Authentication.
- Event Creation: Allow admins to create new events with details like title, date, description, and image.
- Event Deletion: Allow admins to delete events from the database.
- Event Storage: Store event data in Firestore.
- Homepage Display: Display upcoming events on the homepage.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F0F2F5) to ensure readability and a clean aesthetic.
- Accent color: Orange (#FF9800) to highlight key actions and important information.
- Body and headline font: 'PT Sans' for a modern, humanist feel that is easy to read.
- Use clear and consistent icons from Material Design to represent different event categories.
- Employ a card-based layout for event listings on the homepage for a visually appealing and organized display.
- Use subtle animations like fade-ins or transitions when events are added or removed from the homepage to provide visual feedback.
